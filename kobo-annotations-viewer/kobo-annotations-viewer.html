Alternative Method: Manual SQLite Viewer

If the automatic parsing doesn't work, you can use these steps:

  1. Download and install DB Browser for SQLite
  2. Open your KoboReader.sqlite file with it
  3. Go to the "Execute SQL" tab
  4. Run this query to get your annotations: 
    SELECT 
    c.Title,
    c.Attribution as Author,
    b.Text as Highlight,
    b.Annotation as Note,
    b.DateCreated,
    CASE 
        WHEN b.Type = 0 OR b.Type = 1 THEN 'Highlight'
        WHEN b.Type = 2 THEN 'Bookmark'
        ELSE 'Note'
    END as Type
FROM Bookmark b
JOIN content c ON b.VolumeID = c.contentID
WHERE b.Text != '' OR b.Annotation != ''
ORDER BY c.Title, b.DateCreated;
  5. Export the results as CSV or HTML

How to find your Kobo database file

  1. Connect your Kobo eReader to your computer via USB
  2. Open the Kobo drive on your computer
  3. Navigate to the .kobo folder (note: it may be hidden on some systems)
  4. Look for the file named KoboReader.sqlite
  5. Select that file in the upload area above
  6. Note: This tool runs entirely in your browser. Your data never leaves your computer.
